Need for speed

Posted on February 3, 2022

I remember that when I start using ConTeXt almost 20 years ago, one of the annoyances with ConTeXt was that compilation was slower compared to LaTeX. I regularly use both LaTeX and ConTeXt, and these days both are comparable.

I haven’t done any controlled experiments, but here is a result for two medium sized documents that I am working on:

  • A research paper written in LaTeX. It is like a standard math paper, 25 pages, 8 pre-gerenrated PDF plots (generated using PyPlot in Julia), microtype enabled, using acmart class, and using additional packages mathtools, amsfonts, dsfont, algorithm, algorithmic, and subcaption. On a clean directory, compiled via pdflatex engine (version 3.141592653-2.6-1.40.23), and using latexmk for full compilation (which takes 3 runs):

    $time latexmk -pdf paper.tex

[....]
latexmk -pdf paper.tex  42.57s user 1.29s system 99% cpu 43.958 total

  • Solutions for an exam written in ConTeXt. It is 16 pages, generates a few figures using TikZ (figures are not cached), includes a few plots generated using PGFplots with data generated using Octave (the octave results are cached using the filter module). On a clean directory, compiled via luametatex engine (level 20221202), and using the default context running for full compilatin (which takes 2 runs):

    $context exam.tex

[....]
system          | total runtime: 18.586 seconds of 18.827 seconds

So, on average, LaTeX is processing 1.76 pages per second, while ConTeXt is processing 1.72 pages per second. For both system, the time calculation includes the time spent by the helper scripts to resolve references etc. for the full run. Note that this is not an apples to apples comparison. The LaTeX document is just math + pre-generated figures, while the ConTeXt document extensively uses TikZ/PGFplots and external programs.

For LaTeX, figure inclusion appears to be the bootleneck; for ConTeXt, pgfplots appear to be the bottleneck. But, as a user, I don’t care what the bottleneck is as I need these features in the respective documents.
